# Tower Defense Game

---

# 🇺🇸 English

## Overview

Tower Defense is a modern desktop tower defense game developed with Python and PyQt6. Players must strategically place towers, defend against enemy waves, upgrade defenses, and survive through increasingly difficult stages.

## Features

- Modern graphical interface using PyQt6
- Dark and Light themes
- Multilingual support
  - English
  - فارسی
  - 中文
- Multiple tower types
  - Arrow Tower
  - Cannon Tower
  - Laser Tower
  - Freeze Tower
  - Missile Tower
- Tower upgrades
- Tower selling system
- Multiple enemy classes
- Progressive difficulty scaling
- Score and gold system
- Particle effects and animations
- Pause and resume support
- Victory and Game Over screens

## Requirements

- Python 3.10 or newer
- PyQt6

## Install Required Libraries

### Windows

```bash
pip install PyQt6
```

### Linux

```bash
pip3 install PyQt6
```

### macOS

```bash
pip3 install PyQt6
```

## Run

```bash
python "Tower Defense.py"
```

## Gameplay

1. Start the game.
2. Select a tower from the side panel.
3. Place towers on valid tiles.
4. Defeat enemies to earn gold.
5. Upgrade towers to increase power.
6. Survive all waves and achieve victory.

Tower Defense 是一个使用 Python 和 PyQt6 开发的桌面塔防游戏。玩家需要建造和升级防御塔，抵御不断增强的敌人并完成所有关卡。

## 功能特点

- 基于 PyQt6 的现代图形界面
- 深色与浅色主题
- 多语言支持
  - English
  - فارسی
  - 中文
- 多种防御塔
  - 箭塔
  - 炮塔
  - 激光塔
  - 冰冻塔
  - 导弹塔
- 防御塔升级系统
- 出售防御塔功能
- 多种敌人类型
- 难度逐步提升
- 分数与金币系统
- 粒子特效与动画
- 暂停与继续功能
- 胜利与失败界面
